## 🐛 问题描述

**现象：** 选择目录时只能选择根目录下的文件夹，无法保存到子目录

**原因：**
1. 下拉列表只显示当前路径的文件夹
2. 没有文件夹导航功能
3. 无法进入子目录并选择

---

## 🔧 修复方案

### 1. 新增文件夹浏览器 UI

**功能：**
- 📂 可视化文件夹列表
- 🔙 返回上级目录按钮
- ✓ 选中状态标记
- 👆 单击选择，双击进入
- ⌨️ 键盘导航支持

---

### 2. 交互设计

**操作方式：**
```
单击文件夹     → 选择此文件夹
双击文件夹     → 进入此文件夹
点击返回按钮   → 返回上级目录
点击根目录     → 选择根目录
```

**键盘支持：**
```
Enter 键  → 进入文件夹
Space 键  → 选择文件夹
Tab 键    → 切换焦点
```

---

### 3. 视觉设计

**选中状态：**
```scss
&.selected {
  background: rgba(0, 123, 255, 0.08);  // 浅蓝色背景
  border-color: #007bff;                 // 蓝色边框
}
```

**勾选标记：**
```scss
&__check {
  color: #28a745;  // 绿色对勾
  font-size: 18px;
}
```

**当前选择显示：**
```

```
