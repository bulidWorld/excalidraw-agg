## 📖 使用教程

### 保存绘图

1. **绘制图形**
   - 使用工具栏绘制任意图形

2. **点击保存按钮**
   - 点击右下角"导出"按钮
   - 选择"保存到服务器"卡片

3. **选择保存位置**
   - 在"保存绘图"标签页
   - 从下拉列表选择文件夹
   - 或点击 ➕ 创建新文件夹

4. **输入文件名**
   - 默认自动生成文件名
   - 可以手动修改

5. **点击保存**
   - 点击"💾 保存"按钮
   - 等待保存成功提示
   - 对话框自动关闭

### 浏览文件

1. **切换到浏览模式**
   - 点击"📁 浏览文件"标签页

2. **导航文件夹**
   - 点击文件夹进入
   - 点击返回按钮回到上级

3. **查看文件信息**
   - 文件名
   - 文件大小（KB/MB）
   - 最后修改时间

## 📊 文件管理

### 查看已保存的文件

**方法 1: 通过 UI**
- 打开保存对话框
- 切换到"浏览文件"标签页
- 浏览文件夹和文件

**方法 2: 通过文件系统**
```bash
ls -lht /opt/apps/excalidraw-agg/data/
```

### 创建文件夹

1. 打开保存对话框
2. 点击 ➕ 按钮
3. 输入文件夹名称
4. 按 Enter 或点击"创建"

### 文件命名规则

- 支持中文
- 支持特殊字符（自动替换）
- 自动添加 `.excalidraw` 扩展名
- 建议使用有意义的名称

## 🐛 故障排查

### 问题 1: 看不到新 UI

**解决方案：**
1. 强制刷新浏览器 (Ctrl+Shift+R)
2. 清除浏览器缓存
3. 检查浏览器控制台是否有错误

### 问题 2: 保存失败

**检查步骤：**
1. 确认已选择文件夹
2. 确认文件名不为空
3. 查看错误提示信息
4. 检查后端日志

**查看日志：**
```bash
tail -f /tmp/excalidraw-server.log
```

### 问题 3: 文件列表加载失败

**解决方案：**
1. 检查网络连接
2. 刷新页面重试
3. 查看浏览器控制台错误
4. 检查后端服务状态

**检查服务：**
```bash
curl http://localhost:10514/api/health
```
